# Dental 3D Scanner - Flutter App

A Flutter-based Android application for 3D scanning dental models using ARCore technology.

## Project Structure Explained

This document explains what each file does in beginner-friendly terms.

## Getting Started

### Prerequisites
- Flutter SDK installed on your computer
- Android Studio installed
- An Android device with ARCore support (Android 7.0+)

### Installation Steps
1. Install Flutter: Follow the official guide at https://flutter.dev/docs/get-started/install
2. Install Android Studio: Download from https://developer.android.com/studio
3. Set up an Android device or emulator
4. Open this project in Android Studio or your code editor
5. Run `flutter pub get` to install dependencies
6. Run `flutter run` to launch the app
